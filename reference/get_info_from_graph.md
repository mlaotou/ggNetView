# Extract node and edge information from graph object

Extract node and edge information from graph object

## Usage

``` r
get_info_from_graph(graph_obj)
```

## Arguments

- graph_obj:

  An graph object from build_graph_from_mat or build_graph_from_df. The
  network object to be visualized.

## Value

a list
